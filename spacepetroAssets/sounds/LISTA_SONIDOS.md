# Sonidos para Space Shooter

Coloca estos archivos **en esta misma carpeta**:  
`juego disparos rayib/assets/sounds/`

| Archivo        | Uso                          | Formato | Notas                    |
|----------------|------------------------------|--------|---------------------------|
| **shoot.wav**  | Disparo de la nave           | WAV    | Corto, ~0.1–0.3 s         |
| **explosion.wav** | Enemigo destruido por bala | WAV    | Corto, impacto/explosión  |
| **player_hit.wav** | Enemigo te toca (pierdes vida) | WAV | Corto, daño               |
| **game_over.wav** | Pantalla de Game Over       | WAV    | Corto o frase             |
| **music.ogg**  | Música de fondo (opcional)   | OGG    | Se repite en loop         |

- **Efectos**: WAV, cortos (menos de 1 s recomendado).
- **Música**: OGG recomendado para fondo; si no añades `music.ogg`, el juego funciona igual sin música.

Si falta algún archivo, el juego no se rompe: ese sonido simplemente no se reproduce.
